# ChatIcon

Speech-bubble SVG used as the TGV Chat visual wherever we need a neon-green chat icon.

## Source

- File: [../icons/ChatIcon.tsx](../icons/ChatIcon.tsx)

## Import

```tsx
import { ChatIcon } from "./icons";
```

## Props

- `size?: number` — width + height in px (default `14`)
- All standard `SVGProps<SVGSVGElement>`

Color is inherited via `currentColor`.

## Current usages

- [ChatDrawer](ChatDrawer.md) — drawer tab + header title
- [ChatDrawer](ChatDrawer.md) — sidebar "TGV Chat" pinned row
