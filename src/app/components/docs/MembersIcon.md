# MembersIcon

Multi-person silhouette. Used wherever "members" or "online users" are represented as a single visual affordance.

## Source

- File: [../icons/MembersIcon.tsx](../icons/MembersIcon.tsx)

## Import

```tsx
import { MembersIcon } from "./icons";
```

## Props

- `size?: number` — width + height in px
- All standard `SVGProps<SVGSVGElement>`

## Current usages

- [TopNav](TopNav.md) — "N online" collapsed DDM pill (when >2 logged-in users)
- [ChatDrawer](ChatDrawer.md) — header "N online" collapsed DDM pill (mirrors TopNav)
- [ChatDrawer](ChatDrawer.md) — Group rows in the sidebar list
